oracle-hci.txt = McKesson Horizon Clinical Infrastructure, also known as McKesson HCI, utilizes hardcoded passwords for Oracle database access. HCI serves as the patient record datastore for the majority of McKesson applications. There are two components to an HCI implementation: the Infrastructure (or Master) server and the database back-end. The HCI Infrastructure Server has an Oracle client installed that initializes OCI/sqlplus connections to the Oracle database back-end.
